You're right — `abyss.stats()` and `abyss.depth()` don't change the page at all. They're **console-only** — they just print information back to you *inside* the DevTools console itself. That's intentional. Think of it like a secret command-line interface for the page.

Here's exactly what each does:

**`abyss.stats()`**
Prints your viewing statistics right there in the console:
```
📊 Cinema Abyss · Viewing Stats

Watched:  12 / 133  (9.0%)
First:    Film #3
Most recent: Film #47
```
It reads from your `localStorage` — so it knows exactly which films you've marked watched and in what order.

**`abyss.depth()`**
Maps your progress onto the depth of the **Mariana Trench** (11,000 metres — the deepest point on Earth) and prints how far you've descended:
```
🌊 Depth Report

You are at 993m below the surface.
The Mariana Trench is 11,000m deep.
Remaining: 10,007m
```
Watch all 133 films and it prints:
```
✦ You have reached the bottom. There is no bottom.
```

**`abyss.light()`**
This one *does* affect the page — you already found it. It strips everything except film titles and directors. Pure reading mode, no descriptions, no placements, no checkboxes. Run it again to restore everything.
